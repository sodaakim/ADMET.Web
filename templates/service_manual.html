Introduction

This manual is designed for users of the ADMET web service. ADMET is a web-based platform that assists in predicting the absorption, distribution, metabolism, excretion, and toxicity of drugs.

Service Overview

The ADMET service consists of two main functionalities

  • Analyze Page : Enter a single SMILES string to analyze the ADMET characteristics of the corresponding compound.
  • Screening Page : Enter multiple SMILES strings in batch to analyze the ADMET characteristics of several compounds.

Before You Start

  • You must have the SMILES (Simplified Molecular Input Line Entry System) strings prepared.
  • If the SMILES structure is incorrect, analysis will not proceed. For screening, analysis will proceed only for possible structures.
  • An internet connection is required.

How to Use the Analyze Page

  1. Navigate to the Analyze page of the web service.
  2. Enter the SMILES string of the compound you wish to analyze in the 'Enter SMILES' field.
  3. Use the example button to input a sample molecule.
  4. To start the analysis, click the 'Start Analysis' button.
  5. Wait for the processing to complete, then you can check the analysis results on the results page.

How to Use the Screening Page

  1. Navigate to the Screening page of the web service.
  2. Input the SMILES strings of various compounds, separated by line breaks.
  3. Use the example button to input sample molecules.
  4. You can add files to input SMILES.
  5. To start the analysis, click the 'Start Screening' button.
  6. Wait for the processing to complete, then you can check the analysis results for each compound on the results page.

Interpreting Results

Results page displays detailed information and prediction results for each compound's ADMET characteristics. It explains the significance and importance of each indicator.

  • Analyze Results: You can download the results in CSV or PDF format.
  • Screening Results: You can download the content for all molecules in CSV format. You have the option to select the download option. You can check detailed information for individual molecules using the “detail” button.
